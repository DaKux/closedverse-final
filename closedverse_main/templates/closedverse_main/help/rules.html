Closedverse Rules

This page used to only have one rule, but since more problems have risen with only one rule, we have expanded it.

Yeah, this page might be long, and it might be strict, but this isn't the Miiverse Code of Conduct.

NSFW

At the moment, NSFW stuff is forbidden, unless in messages where the other person is okay with it.
We might be implementing an "NSFW community" system, in which someone can post there and not have the posts their in their profile, and also viewing the community will give a warning. If this is implemented, this rule will be edited (not implemented as of October 19)

Offensive content

It isn't okay if a lot of people can see it, or if it's directed at a person/group. This includes offending furries.

Porn/gore

A lot of people don't want to see it. Please do not do that. If you do, lots of users will be happy to snitch you. You will not have any mercy laid on you.

I had to address this too

Please do not say stuff like "ALLAHU AKBAR" or what have you. I'm being serious. Do that on any other forum or well moderated place and you'll be penalized if you do. It's not a "that's offensive to my culture" thing, content like this is removed on lots of sites for a reason. I'm in trouble if you are, alright? I worked hard on this..

Hate speech

This is strongly forbidden. If we can't talk you out of stopping, then no mercy will be laid on you. If you want a free speech allowing social network, please try GAB. We are NOT welcome to hate and hate speech, and while you might be reasoning with your "free speech", we are very diverse, and that diversity can be easily offended. Yes, there exist people that can be offended.

Unwelcome sexual advances

Unless you're joking, it's disgusting, please don't. If you are, then you probably don't know what this is.

Hacking

I'm pretty hack-proof. You don't need to hack me to see my source code, it's available at https://github.com/ariankordi/closedverse.
If you can't read that, perhaps you should spend your time figuring out how to.
Specifically, I'm using Django. It's free of 99.9999% of injections, so there isn't any point to injecting HTML or SQL. If you do, you'll look like an idiot.

Spam

Please don't, I have an Amazon Web Services DigitalOcean bill to pay at the end of each month.

Yeahing

This is spam, unless it's legit. We'll let you know if you are doing this, and if you don't learn from any staff telling you to stop, you're not going to be able to yeah ;)
Yeah scripts are strictly forbidden. If you use one, prepare to die.

Public Mii/Mii Police/12 accounts/impostoring popular Miiverse users

You're not funny, and you won't have any mercy laid on yourself. You'll know if you're doing this.


Repeals

Unless you're too bad, you can get more chances. If we still don't like your behavior, you'll be returned to the state you were.

Begging for an unban

If you only do it in one session, you'll be forgiven. If you do not stop begging an admin for an unban on a separate messaging site, you will be ignored unless you write a beautifully structured repeal asking why you should have an unban.



Age

Most of the time, it's just a number. We don't explicitly allow users under 13 to join, but if you are found being immature and you are under 13 (or 12), you'll be notified. If You Type Like This, You Will Probably Be Dinged As An Example.

Names/branding

I would not advise making anything fanmade for this or about this without vague approval.

What to call this

If you do make any fanmade content for or about this, don't expect it to be alive, and be ready for it to be taken down by me only because I don't want that many stuff like that for this, and if I do then it'll be official and not user content.
